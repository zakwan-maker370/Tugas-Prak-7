RUANG EKSPRESI
Wadah Kreatif & Kompetitif Mahasiswa

RUANG EKSPRESI adalah platform yang dirancang untuk mendukung mahasiswa dalam mengembangkan kreativitas, berkompetisi secara sehat, dan memamerkan karya terbaik mereka. Proyek ini bertujuan memberikan ruang kolaboratif dan inovatif melalui fitur unggulan seperti upload karya, kompetisi, komunitas, serta dashboard personal.
